IMC

L'IMC, ou Indice de Masse Corporelle, est un indicateur de santé qui évalue la relation entre le poids et la taille d'une personne. Il permet de classer une personne en catégories telles que sous-poids, poids normal, surpoids ou obésité.

J2D

J2D est un jeu de devinette en Python où l'ordinateur sélectionne un nombre entre 1 et 10. Le joueur doit deviner le nombre en utilisant les indices fournis par l'ordinateur sur si le nombre est plus élevé ou non, avec une limite de 3 tentatives

F_MDP

F_MDP (Fiabilité de Mot De Passe) est un projet Python visant à évaluer la robustesse d'un mot de passe, permettant de déterminer s'il est sécurisé ou vulnérable.

TDM

Le programme Python TDM (Tableau De Multiplication) a été conçu dans le but de fournir à l'utilisateur la table de multiplication de son choix de manière personnalisée.
